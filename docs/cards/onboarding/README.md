# 온보딩 카드 문서

이 폴더는 실제 온보딩 카드 제작을 위한 카드별 원천 문서다.

## 확정 용어

- `병력`: 카드 오른쪽 위 검은 원에 적는 수치이자, 카드 아래에 놓이는 점령/생존/소모 단위다. 모든 카드의 소모/생존/점령 단위 표현은 `병력`으로 통일한다.
- `명령`: 대장군이 자기 차례에 한 번 사용할 수 있는 리더 능력이다.
- `전투`: 하나의 유닛과 유닛 사이에서 일어나는 한 번의 싸움이다.
- `교전`: 전투 중인 병사 또는 장군들이 상대의 공격범위에 엮인 클러스터다. 교전은 페이즈가 아니라 전장 상태다.
- `보류`: 군영의 카드를 뒷면으로 놓고 다음 보급 때 그 수만큼 추가 보급을 받는 행위다.
- `매복`: 뒤집어서 지형에 둘 수 있는 카드다.
- `매장지`: 사용된 카드 및 전사한 카드가 가는 플레이어별 개인 영역이다.
- `전사`: 카드가 전장에서 제거되어 매장지로 가는 것이다.
- `단말마`: 카드가 전사했을 때 발동하는 키워드다.
- `자폭`: 공격범위 내 상대 병사가 있을 때 자신을 전사시키고 상대 병사를 전사시키는 키워드다.
- `귀신`: 종족이다. 카드 효과로 이번 차례 동안 임시로 얻을 수도 있다.

## 공통 규칙

- 군영의 최대 카드 수는 10장이다.
- 교전은 퇴각 또는 전멸로 끝난다.
- 교전에서 승리한 플레이어는 해당 교전에서 이긴 병사 수만큼 점령 포인트를 얻고, 즉시 점령카드를 배치할 수 있다.
- 숫자 키패드 기준 범위는 카드에 적힌 값을 그대로 따른다. `5`가 자기 칸인지 여부도 카드별 표기를 따른다.
- 도구는 병사에게 장착하거나 필드 위 장악범위 내 칸에 배치한다.
- 도구가 놓인 칸에는 점령카드를 함께 놓을 수 없다.
- 병사 카드는 도구가 놓인 칸에 함께 놓일 수 있다.
- 사용되거나 제거된 도구는 매장지로 간다.
- 초혼으로 배치한 병력 토큰은 이번 차례 동안 장악만 할 수 있고, 차례가 끝나면 사라진다.
- 대장군은 필드에 일반적으로 나올 수 없는 리더 카드다.
- 플레이어는 장군과 별도로 대장군 1장을 함께 사용한다.
- 대장군은 자기 차례에 한 번 명령을 사용할 수 있다.

## 구성

- 중립 병사: 5종, 각 3장, 총 15장.
- 중립 도구: 3종, 각 3장, 총 9장.
- 중립 작전 문서: 2종, 각 3장, 총 6장.
- 캐릭터 특성 병사/장군: 클래스별 총 6장.
- 캐릭터 특성 작전 문서와 도구: 클래스별 2종, 각 2장, 총 4장.
- 대장군: 예맥, 사로국, 가락, 십제 각 1장.
