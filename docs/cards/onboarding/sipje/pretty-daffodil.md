# 어여쁜 수선화

- 카드 종류: 병사
- 진영/클래스: 십제
- 수량: 2장
- 종족: 귀신
- 타입: 병사
- 키워드: 출정
- 병력: 0
- 장악범위: 7,9
- 장악범위 사람 아이콘: 5
- 공격범위: 7,9
- 공격범위 사람 아이콘: 5

## 일러스트 방향

악귀 형태의 일러스트. 걸어다니는 식인꽃 같은 느낌.

## 효과

`[출정]` 내 장악범위 위의 인간을 전사시킨다. 해당 인간이 갖고 있던 병력만큼 이 병사의 병력 +N.

## 온보딩 역할

십제의 귀신 병사가 인간을 제물로 삼아 병력을 얻는 구조를 보여준다.
