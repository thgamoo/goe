# 십제 온보딩 카드

십제 온보딩 카드는 인간, 귀신, 매장지, 단말마, 임시 병력 토큰을 중심으로 한다.

## 구성

- [검은 상여꾼](./black-bier-carrier.md): 매장지에서 병사를 다시 출정시키는 인간 병사.
- [어여쁜 수선화](./pretty-daffodil.md): 인간을 전사시키고 병력을 흡수하는 귀신 병사.
- [다루태자](./prince-daru.md): 전사 시 덕좌태자를 불러오는 인간 장군.
- [덕좌태자](./prince-deokjwa.md): 출정 시 다루태자를 기지 위에 올리는 귀신 장군.
