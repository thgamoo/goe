# 퇴각하라!

- 카드 종류: 작전 문서
- 진영/클래스: 중립
- 수량: 3장
- 타입: 작전 문서
- 키워드: 전령

## 효과

내 병사 하나를 지정한다. 해당 병사와 해당 병사와 이웃한 모든 병사를 상대의 공격 영향범위 밖으로 이동시킨다. 이동시킨 칸 수만큼 내 패를 `전령`으로 보낸다. 전령으로 보낼 수 없다면 이동할 수 없다.

## 전령 설명

군영의 카드를 뒤집어 보내고, 다음 보급 시 뒤집어진 카드 수만큼 추가로 보급하는 행위다. 군영 최대 장수는 10장이기 때문에 전령으로 추가로 챙길 수 있는 장수는 한정되어 있다.

## 온보딩 역할

퇴각으로 공격 영향범위에서 벗어나는 방법과 전령 비용을 가르친다.

## 확인 필요

- `전령 없이 공격`처럼 전령 비용을 무시하는 효과와의 관계.
