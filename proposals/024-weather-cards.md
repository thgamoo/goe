# Proposal 024: 날씨 카드 10종 세팅과 이미지 생성

## 배경

날씨 카드는 기본 날씨 6종과 세력 특수 날씨 4종으로 구성한다.

이번 작업은 사용자가 지정한 카드명, 효과, 플레이버 텍스트를 프로젝트 카드 데이터와 화면에 반영하고, 이후 각 날씨 카드에 맞는 이미지 생성 프롬프트 또는 실제 이미지 에셋을 준비하는 것이다.

## 확정 카드 목록

| 카드명 | 효과 | 플레이버 텍스트 |
| --- | --- | --- |
| 창천 | 효과없음 | 맑은 날씨네요 |

## 구현 방향

승인 후 다음을 진행한다.

1. `task.csv`가 없으면 생성하고, 이번 작업 행을 추가한다.
2. `src/data/cards.json`에 `weather` 또는 `weathers` 섹션을 추가한다.
3. 날씨 카드 타입을 앱 타입 정의에 추가한다.
4. 카드 DB 화면에서 날씨 카드가 검색/필터/카드 렌더링에 포함되도록 반영한다.
5. 필요하면 날씨 카드 전용 탭 또는 필터를 추가한다.
6. 이미지 생성은 카드별 분위기가 겹치지 않도록 프롬프트를 만든 뒤 생성한다.

## 이미지 방향

카드 일러스트는 기존 괴이전 카드 톤에 맞춰 어둡고 거친 붓질, 고대 전장, 판타지 재난의 분위기를 사용한다.

- 창천: 맑은 하늘과 정돈된 전장
- 비: 젖은 깃발, 진흙이 아니라 빗줄기와 장비 부담 중심
- 눈: 눈발 속 대치와 차가운 은폐
- 안개: 시야가 끊기는 회백색 전장
- 폭풍: 휘몰아치는 비바람, 무너진 깃발, 점령선이 흔들리는 전장
- 혹한: 얼어붙은 병장기와 서리 낀 진영
- 적무: 붉은 안개와 광기 어린 전투 기운
- 백야: 창백하게 밝은 밤, 길과 그림자가 어긋나는 도깨비 현상
- 피안: 이승과 저승의 경계, 귀신과 함께 무너지는 전장
- 과열: 달아오른 지면, 갈라진 용암, 철기를 제외한 병사가 버티기 어려운 열기

## 확인 질문

1. 날씨 카드는 카드 DB에 함께 섞어 보여줄까요, 아니면 `날씨카드` 별도 탭을 만들까요?
2. `비: 이동 시 [보류 1]`, `눈: 공격 시 [보류 1]`은 해당 행동을 하려면 보류 1을 비용으로 내는 의미가 맞나요?
3. `피안: 귀신과 전투 종료 시 귀신과 함께 전사`에서 전사하는 대상은 귀신과 전투한 상대 병사 1장인가요, 아니면 해당 전투에 참여한 모든 비-귀신 병사인가요?
4. `과열`의 `철기`는 병사 `species`, `className`, `keywords` 중 어느 필드로 판정할까요?
5. 이미지 생성 결과는 10장 모두 개별 파일로 저장하는 방향이 좋을까요, 아니면 먼저 콘셉트 시안 1장으로 톤을 맞춘 뒤 나머지를 만들까요?
