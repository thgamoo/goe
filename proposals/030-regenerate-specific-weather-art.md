# Proposal 030: 특수 날씨 카드 5종 일러스트 재생성

## 배경

기존 날씨 카드 10종 중 특수 날씨 5종의 장면성을 강화한다.

이번 작업은 카드 데이터의 이름과 효과는 유지하고, 지정된 이미지 파일 5개만 새 이미지로 교체하는 것이다.

## 대상 카드

다음 파일을 새 이미지로 교체한다.

| 카드명 | 파일 | 새 이미지 기준 |
| --- | --- | --- |
| 혹한 | `public/card-assets/weather/weather-bitter-cold.png` | 얼어붙은 사람이 깨지는 일러스트 |
| 적무 | `public/card-assets/weather/weather-red-mist.png` | 눈에서 붉은 안광이 나오면서 병사들을 도륙하는 실루엣 |
| 백야 | `public/card-assets/weather/weather-white-night.png` | 외뿔도깨비들이 돌아다니고 탈들이 바닥에 널려있는 일러스트 |
| 피안 | `public/card-assets/weather/weather-beyond-shore.png` | 피안화 꽃들이 바닥에 흐드러지게 피어 있고, 해골에 꽃이 피면서 쓰러지는 일러스트 |
| 과열 | `public/card-assets/weather/weather-overheat.png` | 땅이 갈라진 용암 사이로 병사들이 빠져서 허우적대는 일러스트 |

## 이미지 방향

공통 방향:

- 기존 괴이전 카드와 어울리는 세로형 카드 일러스트
- 고대 전장, 어두운 동양 판타지, 먹선과 거친 붓질
- 카드 프레임과 텍스트가 얹히므로 테두리와 글자는 넣지 않는다.
- 하단 20%는 효과 텍스트가 읽힐 수 있도록 지나치게 복잡하지 않게 둔다.
- 직접적인 과잉 고어 표현은 피하고, 실루엣/상징/파편/동작으로 위험을 읽히게 한다.

## 구현 방향

승인 후 다음을 진행한다.

1. `task.csv`에 이번 작업 행을 추가한다.
2. 이미지 생성 도구로 5개 이미지를 각각 생성한다.
3. 생성 결과를 기존 파일명에 맞춰 `public/card-assets/weather/`에 복사한다.
4. `pnpm build`를 실행한다.
5. 브라우저에서 카드 DB의 날씨 필터를 확인하여 이미지가 깨지지 않는지 검증한다.
6. 작업이 끝나면 `task.csv` 상태를 `done`으로 갱신한다.

## 확인 질문

1. 기존 5개 파일을 그대로 덮어써도 될까요, 아니면 `-v2` 파일로 저장하고 카드 데이터 경로를 바꿀까요?
2. `적무`의 “도륙”은 피가 튀는 직접 장면보다 검은 실루엣과 붉은 안광 중심의 연출로 처리해도 될까요?
3. `백야`의 외뿔도깨비는 귀엽지 않고 불길한 전장 요괴 느낌으로 갈까요?
4. `피안`의 꽃은 붉은 피안화 중심으로, 보라/회색 안개를 보조 색으로 써도 될까요?
